Display: inline

Muestra el elemento en línea, una manera de entenderlo sería "En el mismo renglón" al que no se le pueden apicar valores para cambiar sus dimensiones, además éste no afecta los elementos contiguos. Muchas etiquetas en HTML vienen por defecto con éste valor, mira la lista aquí.

Display: block

Muestra el elemento como bloque, esto quiere decir que sin importar su tamaño abarca toda la linea, en otras palabras "Ocupa todo el renglon." Por lo tanto los elementos contiguos se muestran en la siguiente línea. Al igual que el valor anterior, muchas etiquetas en HTML vienen por defecto con ésta característica entra aquí para conocer la lista.
