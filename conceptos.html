Display

La propiedad CSS display especifica si un elemento es tratado como block or inline y el diseño usado por sus hijos, como flow layout(Diseño de Flujo), grid(Cuadricula) o flex(Flexible). (MDN web docs, 2020)

Cada elemento HTML tiene un valor de visualización predeterminado según el tipo de elemento que sea. El valor de visualización predeterminado para la mayoría de los elementos es block o inline. (w3schools, 2020)
